# Sitio Web — Requisitos cumplidos

Este proyecto cumple con los siguientes puntos solicitados:

- Todo el sitio está dentro de un contenedor.
- Header con un div que incluye un logo alineado a la izquierda.
- En el body se incluye un enlace a un video de un servicio de streaming.
- Footer con la información solicitada (curso, nombre, código, correo).
- Imagen de fondo aplicada a toda la página.

Estructura básica del proyecto:

index.html
assets/
├── logo.png
└── bg.jpg

El sitio puede visualizarse abriendo `index.html` en cualquier navegador y puede desplegarse en plataformas para hosting estático (ej. GitHub Pages, Render Static).
